# Pokedex CLI

A simple CLI Pokedex built in Go using PokeAPI.

## Features

- List locations
- Explore areas
- Catch Pokemon
- Inspect caught Pokemon
- Caching layer

## Installation

git clone https://github.com/lnrascal/pokedex
cd pokedex
go run ./cmd/pokedex

## Example

Pokedex > help
Pokedex > map
